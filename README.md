# React-Native-communication（native和react通信）

#安装与运行

  1.打开终端cd定位到ReactComponent文件夹下，执行npm install命令，安装下载react依赖库，
  
  2.用Xcode打开lizhongfei_rn_test.xcworkspace，运行即可！
